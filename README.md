# WalletMind — Verifiable AI DeFi Copilot for Hedera

Personalized trading, staking, yield, and risk insights for your wallet, with every recommendation immutably logged on‑chain.

## What it does
WalletMind is an AI-powered DeFi copilot that provides a personalized experience for Hedera wallet owners. It analyzes your portfolio across the Hedera network, identifying opportunities for yield, staking, and risk management. Every recommendation and strategy generated by the AI is immutably logged on-chain using the Hedera Consensus Service (HCS), ensuring transparency and verifiability.

**HashScan**: [https://hashscan.io/testnet/topic/0.0.8329089](https://hashscan.io/testnet/topic/0.0.8329089)

## Monetization
0.1–0.3% fee on yield generated via strategies executed through WalletMind (paid in HBAR).

## Key Features
- **Verifiable AI Recommendations**: Every strategy is logged on the Hedera Consensus Service (HCS) for permanent, auditable proof of the AI's logic.
- **Real-Time Portfolio Risk & Yield**: Instant analysis of HBAR balances and fungible tokens via Hedera Mirror Node.
- **Personalized DeFi Strategist**: Uses LangChain and Groq (Llama 3.3 70B) to simulate complex DeFi moves tailored to your specific wallet risk profile.
- **Native Hedera Integration**: Direct connectivity with Hedera network tools, exchange rates, and protocol data.

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS (Vanilla CSS components)
- **AI Engine**: LangChain, Groq (Llama 3.3 70B)
- **Blockchain**: Hedera Testnet (HCS, Mirror Node, Hedera SDK)
